Brief introduction to the project

This project explores the dynamics of a forced damped pendulum, which is a classic problem in physics and engineering. The goal is to understand how the system behaves under various conditions, including different damping coefficients and external forces. The project involves simulating the motion of the pendulum using numerical methods and visualizing the results through graphs and animations.
